**วิดีโอ:**
นำไฟล์วิดีโอมาใส่ในโฟลเดอร์ assets นี้ได้เลย
ให้ตั้งชื่อไฟล์เป็น `video1.mp4`, `video2.mp4`, `video3.mp4` นะครับ
เมื่อกดปุ่ม วิดีโอจะเปิดขึ้นมาในแท็บใหม่ของเบราว์เซอร์

**รูปภาพ:**
รูปภาพจะอยู่ในโฟลเดอร์ `pic` แทนนะครับ
โปรแกรมจะพยายามโหลดรูปภาพตามชื่อไฟล์ที่ระบุไว้ใน `script.js`
คุณสามารถเปิดไฟล์ `script.js` แล้วแก้ไขตัวแปร `imageFiles` เพื่อให้ตรงกับชื่อไฟล์รูปภาพของคุณได้เลย

ตัวอย่างการแก้ไขใน `script.js`:
```javascript
const imageFiles = [
    "รูปฝ้าย1.jpg",
    "รูปฝ้าย2.png", 
    "ทริปทะเล.gif",
    "my-cute-cat.jpeg"
    // เพิ่มชื่อไฟล์รูปภาพของคุณที่นี่
];
```
