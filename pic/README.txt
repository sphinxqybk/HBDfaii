**รูปภาพ:**
นำไฟล์รูปภาพมาใส่ในโฟลเดอร์ pic นี้ได้เลย

โปรแกรมจะพยายามโหลดรูปภาพตามชื่อไฟล์ที่ระบุไว้ใน `script.js`
คุณสามารถเปิดไฟล์ `script.js` แล้วแก้ไขตัวแปร `imageFiles` เพื่อให้ตรงกับชื่อไฟล์รูปภาพของคุณได้เลย

ตัวอย่างการแก้ไขใน `script.js`:
```javascript
const imageFiles = [
    "รูปฝ้าย1.jpg",
    "รูปฝ้าย2.png", 
    "ทริปทะเล.gif",
    "my-cute-cat.jpeg"
    // เพิ่มชื่อไฟล์รูปภาพของคุณที่นี่
];
```

**คุณสมบัติของแกลเลอรี:**
- คลิกรูปเล็กเพื่อดูรูปใหญ่
- ใช้ปุ่มลูกศรซ้าย-ขวา หรือ คีย์บอร์ด (←, →) เพื่อเปลี่ยนรูป
- มีเคาน์เตอร์แสดงตำแหน่งรูปปัจจุบัน (เช่น 1/10)
- อนิเมชันแบบ fade in/out เวลาเปลี่ยนรูป
